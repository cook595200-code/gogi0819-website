# 고기처럼0819 웹사이트

## 프로젝트 개요

고기처럼0819는 서울 강서구 화곡동에 위치한 특수부위 전문 고깃집의 공식 웹사이트입니다. 스타벅스 스타일의 깔끔한 화이트 톤과 고급스러운 다크 그린 계열의 디자인을 사용하여 프리미엄 고깃집의 브랜드 정체성을 표현합니다.

## 🎯 주요 특징

### 디자인 철학
- **스타벅스 스타일 UI/UX**: 깔끔한 화이트 톤과 고급스러운 다크 그린 포인트 컬러
- **프리미엄 감성**: 여백이 많은 레이아웃과 고급스러운 타이포그래피
- **반응형 디자인**: 모든 기기에서 최적화된 사용자 경험
- **미묘한 애니메이션**: 부드러운 페이드인과 패럴랙스 스크롤 효과

### 브랜드 아이덴티티
- **특수제작 숯불 불판**: 묵직함과 뜨거운 숯의 감성
- **희소 특수부위**: 꼬들살·항정살·볼살 등 최상급 부위
- **시그니처 상차림**: 강릉집된장 된장국, 얼음 동동 묵사발, 푸딩 계란찜
- **아늑한 매장 분위기**: 작지만 따뜻하고 프라이빗한 공간

## 📋 구현된 섹션

### 1. 메인 히어로 섹션
- 숯불에 고기가 구워지는 시네마틱 시각적 요소
- "특수부위의 정점을 굿다 — 고기처럼0819" 메인 카피

- 감각적인 문구와 가격 정보

### 3. 불판/조리 철학 섹션

- 불판 철학에 대한 상세 설명

### 4. 시그니처 기본 상차림 섹션
- 강릉집된장 된장국, 얼음 동동 묵사발, 푸딩 계란찜
- 아이콘 기반의 시각적 표현

### 7. 예약 / 문의 섹션
